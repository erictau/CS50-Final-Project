Project Description

This project is a project budget tracking web app that utilizes HTML, CSS, JS, Bootstrap, and ChartJS on the front end, and Python, Flask, and SQLite3 on the back end. This app tracks several budget categories and visualizes the budgets using a bar chart via ChartJS. The user must set the project up initially with task descriptions and dollar amounts. As money is spent out of each budget, the user must add a transaction on the "Transactions". A history of all transactions is found on the "Transactions" page and the budgets shown on the "Home" page are automatically updated for each transaction.

Future Features

  • Include ability to edit and delete transactions
  • Include sorting and filtering of transaction history
  • Add or edit the existing budgets
  • Change the number of budgets from 5 to a user-inputted amount
  • Manage multiple projects simultaneously
  • Save, export, and import projects from Excel and CSV formats.
  • Add subletters for each budget group. For example: A - Mechanical Engineering, A.1 - Mechanical Layout, A.2 - Piping Analysis, etc.
